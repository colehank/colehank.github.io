公式

此主题支持使用 渲染美观的数学公式，支持行内和显示模式。MathJax 3 引擎。你只需要用以下内容包围你的数学表达式 $$，例如 $$ E = mc^2 $$. 如果你将其留在段落中，它将生成一个内联表达式，就像 \(E = mc^2\)。

事实上，您也可以使用单个美元符号 $ 创建内联公式，例如 $ E = mc^2 $, 它将呈现为 $ E = mc^2 $ 。这种方法在基于 TeX 的编译过程中提供了相同的效果，但在视觉上看起来比双美元符号稍微不那么粗。$$，使其更自然地融入周围文本。

要使用显示模式，再次用 将表达式包围起来。$$ 并将其作为单独的段落放置。以下是一个示例：

\[\left( \sum_{k=1}^n a_k b_k \right)^2 \leq \left( \sum_{k=1}^n a_k^2 \right) \left( \sum_{k=1}^n b_k^2 \right)\]

注意 MathJax 3 是 MathJax 的一次重大重写 带来了显著的加载和渲染速度提升。

引用

然后在文章正文中使用引用 <d-cite> 标签。关键属性是参考文献中提供的 id。关键属性可以接受多个 id，用逗号分隔。

引用以内联形式呈现如下： （一个在悬停时显示更多信息的数字）。如果您有附录，则会自动创建并填充参考文献。

Distill 选择了一种数字内联引用样式，以提高引用密集文章的可读性，并且由于悬停时显示更多信息，许多较长引用的好处被消除了。然而，我们认为，如果您详细讨论某些内容并且它很好地融入了文章的流畅性，提及作者的姓氏是一种良好的风格 — 作者是人类，社区将他们与自己的作品联系起来对他们来说是很好的。

脚注

只需将您希望显示在脚注中的文本包裹在 <d-footnote> 标签。脚注的编号将自动生成。这将成为一个可悬停的脚注。

代码块

语法高亮显示提供于 <d-code> 标签。内联代码片段示例：<d-code language="html">let x = 10;</d-code>. 对于较大的代码块，添加一个 block 属性：

var x = 25; function(x) { return x * x; }

注意： <d-code> 块在暗模式下看起来不好。您可以改用标准的 Jekyll 语法高亮与 highlight liquid 标签。

您还可以在三引号中使用带有语言标签的标准 Markdown 代码块，例如：

def foo(x）：
  返回 x

Mermaid

此主题支持直接在 markdown 中创建图表，使用 Mermaid. Mermaid 使用户能够渲染流程图、时序图、类图、甘特图等。只需将图表语法嵌入到 mermaid 代码块中。

要创建甘特图，可以使用以下语法：

```mermaid
gantt
    dateFormat  YYYY-MM-DD
    title 甘特图

    section 部分
    任务 A           :a1, 2025-01-01, 30d
    任务 B           :after a1, 20d
    任务 C           :2025-01-10, 12d
```

以下是它的渲染方式：

gantt
    dateFormat  YYYY-MM-DD
    title A Gantt Diagram

    section Section
    Task A           :a1, 2025-01-01, 30d
    Task B           :after a1, 20d
    Task C           :2025-01-10, 12d

同样，你也可以用它来创建漂亮的类图：

```mermaid
classDiagram

    class Animal {
        +String species
        +int age
```

Diff2Html

此主题还支持集成 Diff2Html，一个工具，可以直接在 markdown 中美观地呈现代码差异（diffs）。Diff2Html 非常适合展示代码更改，清晰地呈现新增、删除和修改内容。它非常适合代码审查、文档和教程，尤其是需要逐步突出代码更改的场景——您甚至可以一次引入多个文件的更改。

Chartjs、Echarts 和 Vega-Lite

Chart.js 是一个多功能的 JavaScript 库，用于创建响应式和交互式图表。支持多种图表类型，如柱状图、折线图、饼图和雷达图，是直接在网页中可视化数据的理想工具。

这是一个 JSON 风格的配置示例，用于在 Chart.js 中创建柱状图：

```chartjs
{
  "type": "bar",
  "data": {
    "labels": ["2017", "2018", "2019", "2020", "2021"],
    "datasets": [
      {
        "label": "Population (millions)",
        "data": [12, 15, 13, 14, 16],
        "backgroundColor": "rgba(54, 162, 235, 0.6)",
        "borderColor": "rgba(54, 162, 235, 1)",
        "borderWidth": 1
      }
    ]
  },
  "options": {
    "scales": {
      "y": {
        "beginAtZero": true
      }
    }
  }
}
```

Vega-Lite 是一种声明式可视化语法，允许用户创建、共享和自定义各种交互式数据可视化。以下 JSON 配置生成了一个简单的柱状图：

```vega_lite
{
  "$schema": "https://vega.github.io/schema/vega/v5.json",
  "width": 400,
  "height": 200,
  "padding": 5,

  "data": [
```

TikZ

TikZ 是一个强大的基于 LaTeX 的绘图工具，由 TikZJax. 你可以轻松地将 TikZ 绘图从论文、海报和笔记中移植过来。例如，我们可以使用以下代码来说明欧拉公式 $ e^{i \theta} = \cos \theta + i \sin \theta $：

<script type="text/tikz">
\begin{document}
  \begin{tikzpicture}
      \filldraw[fill=cyan!10, draw=blue, thick] (0,0) circle (2cm);

      \draw[->, thick] (-2.5,0) -- (2.5,0) node[right] {实部};
      \draw[->, thick] (0,-2.5) -- (0,2.5) node[above] {虚部};

      \draw[->, thick, color=purple] (0,0) -- (1.5,1.5);
      \node[color=purple] at (1.1, 1.7) {$e^{i\theta}$};

      \draw[thick] (0.7,0) arc (0:45:0.7);
      \node at (0.9, 0.3) {$\theta$};

      \draw[dashed, color=gray] (1.5,1.5) -- (1.5,0) node[below, black] {$\cos \theta$};
      \draw[dashed, color=gray] (1.5,1.5) -- (0,1.5) node[left, black] {$\sin \theta$};
      \node at (2.2, 0) [below] {1};
      \node at (0, 2.2) [left] {$i$};
      \node at (1.5,1.5) [above right, color=blue] {$(\cos \theta \, \sin \theta)$};
  \end{tikzpicture}
\end{document}
</script>
